## Description

## ðŸ“Œ Project Overview
This **NestJS-powered application** provides:
- **User authentication & authorization** using JWT.
- **Role-based access control (RBAC)** for Admin & Users.
- **Product management system** with filtering, updating, and deletion.
- **Review system** for products.
- **Email verification & password reset** using Mailer.
- **Secure database management** with PostgreSQL & TypeORM.
- **Automated e2e testing** for API endpoints.
- **API documentation** using Swagger.
- **Middleware logging** to track API requests.

---

## ðŸš€ Technologies Used
- **[NestJS](https://nestjs.com/)** - Scalable framework for backend applications.
- **TypeORM** - ORM for handling PostgreSQL database operations.
- **JWT** - Secure authentication and user session handling.
- **Multer** - Middleware for image uploads.
- **Mailer Module** - Sending verification & password reset emails.
- **Swagger** - API documentation.
- **Helmet.js** - Security middleware for enhanced API protection.
- **Supertest** - Automated e2e API testing.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
